## [0.1.0] - 2025-05-25
### Added
- Initial release of `runefix-core`
- Grapheme-aware display width engine
- Support for emoji, CJK, Kana, Hangul, fullwidth variants
- Integration script for char-table datasets
- Example: `examples/text_align.rs`
